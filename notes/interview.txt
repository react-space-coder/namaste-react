1 Difference between “ == “ and “ === “ operators.
->Both are comparison operators. The difference between both the operators is that “==” is used 
to compare values whereas, “ === “ is used to compare both values and types.
->== will convert the both values to same type and compare the value where as === will compare both
value and type of the value then return
